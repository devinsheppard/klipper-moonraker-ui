# Three Rivers Logo Watermark

The Raiders theme watermark checks for a local logo in `public/branding` and uses the first match:

1. `/branding/three-rivers-college-logo.svg`
2. `/branding/three-rivers-college-logo.png`
3. `/branding/three-rivers-college-logo.webp`

Current file in this repo:
- `public/branding/three-rivers-college-logo.png`

The watermark styling is controlled in `src/styles.css` and is designed for subtle decorative background use (low opacity, masked, non-interactive).
